看了这篇博客知道Fibonacci 有$O(logn)$ 的解法，所以来总结下计算 Nth Fibonacci 的算法。

     int fib(int N) {
        if (N == 0 || N == 1) {
            return N;
        } else {
            return fib(N - 1) + fib(N - 2);

递归算法不推荐，因为时间复杂度为指数级。

可以采用记忆化Memoization 降低复杂度。

      int fib(int N) {
        if (N == 0 || N == 1) {
            return N;
        }
        int f1 = 0, f2 = 1, f = 0;
        for (int i = 2; i <= N; i++) {
            f = f1 + f2;
            f1 = f2;
            f2 = f;
        }
        return f;
    }

时间复杂度：$O(n)$

空间复杂度：$O(1)$

int ExponentiationBySquaring(int a, int n)
{
    int res = 1;
    while(n){
        if(a&1){
            res *= a;
        }
        a *= a;
        n >>= 1;
    }
    return res;
}

 static int[][] matrix_mul(int[][] a, int[][] b) {
        int[][] tmp = new int[2][2];
        for (int i = 0; i < 2; i++) {
            for (int j = 0; j < 2; j++) {
                tmp[i][j] = 0;
                for (int k = 0; k < 2; k++) {
                    tmp[i][j] += a[i][k] * b[k][j];
                }
            }
        }

        return tmp;
    }

    int matrix_quick_exp(int[][] a, int n) {
        int[][] ans = {{1, 0}, {0, 1}};
        while (n != 0) {
            if (n % 2 == 1) {
                ans = matrix_mul(ans, a);
            }
            a = matrix_mul(a, a);
            n >>>= 1;
        }
        return ans[0][0];
    }

    int fib(int N) {
        if (N <= 1) {
            return N;
        }
        int[][] a = {{1, 1}, {1, 0}};
        return matrix_quick_exp(a, N - 1);
    }

4.快速倍增法(Fast doubling)

由$ \begin{bmatrix} 1 & 1 \ 1 & 0 \end{bmatrix}^n = \begin{bmatrix} F(n+1) & F(n) \ F(n) & F(n-1) \end{bmatrix}$

可推出快速倍增法，具体见 mathStackExchange.
